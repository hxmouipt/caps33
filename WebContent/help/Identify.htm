Identify features on the map

If you want to know about a feature on the map, you can get more information by using the identify feature.

To identify a feature

  1. In the Toolbar, click the Identify button. . The button changes to a pressed appearance . (If no Identify button is visible, then the Identify function is not available in this application.)

  2. Click on the feature you want to identify. An entry for the identify results should appear in the Results section of the Console. The result will display the coordinates of the location clicked.

  3. Expand the location by clicking on the plus image to the left of the coordinates, this displays the names of features found.

  4. Click on the feature name to display other attributes of that feature in the Result Details section. You can use the Actions defined on the results to Zoom, Highlight, or Clear Highlight. The same actions are also available by right clicking each result in the Resluts section.

  5. See Using Results for more on using the Results window.
