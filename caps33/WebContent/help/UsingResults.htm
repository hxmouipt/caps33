Using Results

The Results panel displays the output from using the Identify tool, from a search using Find, a Geoprocessing task, or from other operations that output information.

To open the Results window

  • Results panel expands automatically after using any operation that produces a relevant list of results.
  • You can also expand Results by clicking its down-arrow in the Console. If you open Results before doing any Find or Identify, the Results panel will be blank.
  • You can collapse the Results panel to its title bar by clicking the collapse icon.
  • If Results is re-opened, previous results will still be visible, along with any new ones just found.

Navigating the Results

  • After doing a Find or Identify, one or more result items will be added to Results.
  • If you do another Find or Identify, the new results items are added to the top of the list in Results. Previous results remain in the list.
  • To see information associated with a result item, click the plus icon to the left of the result item.
  • When you expand a result item, you will see one or more result item names. Click on the result item to see the detila in the Results Details section.
  • To see details for a feature, click on the name. This displays attribute field names and values for the feature in the Result Details section.

  • Right-click on an item in Results for context menu options, or use the Action links in the Result Details section

  • Features may be highlighted by using highlight option in the context menu, .This causes the feature(s) to be highlighted on the map. By default the highlighting has a yellow outline or fill, but this may be set to a different color by the creator of the map service.
